Standard workflow¶

One of there firts problems I encounter is simply importing SOC module. Currently we have next file structure:

socsim SOC init.py common …. Researchs Notebook1.ipynb Notebook2.ipynb

Official Python documentation suggests next syntax to import from relative path. In current case - module from root folder.

from . import echo
from .. import formats
from ..filters import equalizer

But such syntax fails in Jupyter environment:

[13]:

from .. import SOC

---------------------------------------------------------------------------
ValueError                                Traceback (most recent call last)
<ipython-input-13-02a4c878646c> in <module>
----> 1 from .. import SOC

ValueError: attempted relative import beyond top-level package

[15]:

import os
os.getcwd()

[15]:

'C:\\Users\\ofcra\\dev\\socsim'

Hopefully StackOverflow suggests next simple solution:

[17]:

cd ..

C:\Users\ofcra\dev

Now Jupyter is in the root folder and can easily import SOC module:

[18]:

import SOC
